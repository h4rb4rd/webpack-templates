## Start using for a new project
Clone the repo
Run npm i to install dependencies
Run commands in terminal:

Development Server
```
npm start
```
Production Build
```
npm run build
```
Development mode
```
npm run dev
```
Checking ESLint problems 
```
npm run lint
```
Fixing ESLint problems 
```
npm run lintfix
```
